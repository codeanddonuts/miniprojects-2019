### 사용한 기술

Java, Spring Boot, JPA, Thymeleaf, Websocket, REST-assured, WebTestClient 등

### 기능

- 회원가입
- 글 및 댓글 쓰기, 수정, 삭제
- 글 및 댓글 좋아요
- 친구 맺기
- 프로필 사진 업로드, 글에 사진 첨부
- 이용자 검색
- 알 수도 있는 친구 추천
- 새 글 알림
- 일대일 채팅
